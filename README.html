LiLi影子跟读Pro

专业级语言学习工具 - 百度网盘集成版

功能特色

  • 百度网盘无缝集成，直接访问您的学习材料
  • 支持多种视频格式：MP4、AVI、MOV、WMV、FLV、WebM等
  • 支持多种字幕格式：SRT、VTT、ASS、SSA、SUB、TXT等
  • 智能单词点击查询，内置专业词汇数据库
  • 单句循环练习，提升口语流利度
  • 多种播放速度调节，适应不同学习需求
  • 灵活的字幕显示模式：中英双语、仅英文、仅中文、无字幕

使用步骤

  1. 点击"登录百度网盘"按钮，在弹出的授权页面中完成登录
  2. 登录成功后，您可以看到网盘文件列表
  3. 浏览或搜索您的视频和字幕文件
  4. 点击视频文件的"选择视频"按钮
  5. 点击字幕文件的"选择字幕"按钮
  6. 开始您的语言学习之旅！

重要提示

请确保您的视频和字幕文件格式正确。字幕文件建议使用UTF-8编码，以确保中文字符正确显示。

学习功能

  • 单词点击查询：点击英文字幕中的任意单词查看详细释义
  • 单句循环：重复播放当前句子，计数器显示循环次数
  • 播放速度：调节播放速度（0.25x - 1.25x）
  • 字幕模式：切换字幕显示模式
  • 句句跳转：快速跳转到上一句或下一句

常见问题

  • 登录失败？ 请检查网络连接，确保百度账号密码正确
  • 视频无法播放？ 请检查视频文件是否损坏，或尝试其他视频文件
  • 字幕显示乱码？ 请确保字幕文件使用UTF-8编码
  • 单词点击没反应？ 请确保已加载字幕文件，并且英文字幕正确显示
